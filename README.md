### How To Install
```
$ git clone https://github.com/florienzh4x/Shodan-Search.git Shodan
$ cd Shodan
$ composer install
$ chmod +x shodan
$ ./shodan
```
### Requirements
- PHP 7.0 or latest
- composer
- Member plan shodan
